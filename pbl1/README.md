# PBL 1. 친구 추천

## Given
1. facebook_combined.txt 
> * id1, id2 : id1과 id2는 친구  
> * undirected graph  
2. Similarity threshold: r
> * 0.9, 0.8, 0.7, 0.6  
  
## Goal 
1. 아직 친구가 아닌 모든 사용자 쌍 중에서  
2. 공통되는 친구의 비율이 r이상인 아이디 쌍을 모두 찾으시오.
